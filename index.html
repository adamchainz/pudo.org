Software projects

Current

  • 2013 twindle, client toolkit for the Twitter streaming API, storing and analysing statuses.
  • 2012-2013 dataset, easy-to-use Python library for data processing using relational databases.
  • 2011-2013 OffenesParlament, legislative activity tracking system for both chambers of the German parliament. Features plenary speeches, legislative drafts and personal profiles.
  • 2012-2013 nomenklatura, web-based data cleaning tool to help match aliases of an entity in source data and to map them to its correct form.
  • 2013 databin, a pastebin site for tabular data, enables easy sharing of small data snippets.
  • 2011-2013 ReGENESIS, data extraction from the repository of the German statistical office, data cube reconstruction.

Previous

Set up budget sites for Cameroon, Slovakia, Nepal, Bosnia and Minas Gerais (Brasil). Contributed to Where Does My Money Go? and the data.gov.uk spend browser.
  • 2011-2013 messytables, extracting structured data from badly-formatted tabular data.
  • 2012-2013 LobbyFacts API, an API to provide clean and current data on lobby activity in the European Union, cooperation with Corporate Europe, LobbyControl and Friends of the Earth Europe.
  • 2011 WebStore, RESTful, SQL-based data store and API generator service.
  • 2011 YourTopia, contributor, voting on a custom index for human development criteria. Third price in the World Bank's Apps for Development competition.
  • 2011 FlockScrape, distributed scraping of large web sites, e.g. the German companies register.
  • 2011 DataHub, light-weight, experimental, activity-centric clone of CKAN.
  • 2010 OffenerHaushalt, visualization of scraped German budget data.
  • 2010-2011 CKAN, worked on customizations for PublicData.eu, IATI Registry, Helsinki Regional Infoshare, the Data Hub and various community instances, including OffeneDaten.de in Germany.
  • 2009-2011 Adhocracy, collaborative policy drafting platform for distributed groups. Deployments for the Commission Internet and Digital Society of the German Parliament, the City of Munich and various other organisations.

Datasets

Many of my projects require that I keep up-to-date copies of various public datasets. These are often scraped, or additional information had been added to facilitate analysis. Below is a collection of resources I'm re-publishing because they are significantly enhanced compared to their source form.

European Union

  • Tenders Electronic Daily, the EU's common procurement portal, scraped tenders and contract awards notices from their web site and released data as CSV. For more info, see OpenTED.
  • EC Financial Transparency System, the Commission's direct expenditure (data source, scripts). This version has been geo-coded and reconciled with OpenCorporates.
  • EC/EP Expert Groups, data processing script to transform data about expert groups from XML into a relational database.
  • EC/EP Transparency Register, the API provides up-to-date CSV and JSON exports of a cleaned version of the XML data released by the secretariat. An archive of the register is also being kept here.
  • EU budget, released upon request as XML data, processing scripts will convert this to CSV (2005-2010).

For context, also see the OpenSpending briefing on the finance data sources of the European Union.

Germany

  • Documents, transcripts and other data from the German Bundestag and Bundesrat, scraped on a nightly basis and significantly extended for use in OffenesParlament.
  • German federal budget, scraped from the Ministry of Finance's web site - some of which is now gone. Please note the comments on data quality included in the scraper repository.
  • Granular results for the 2009 federal election, voting district level results data from a CD sold by StaBa, unfortunately doesn't include geographic boundary data.
  • Research funding of eight German ministries, scraped from the Förderkatalog portal.
  • Director's Dealings, German banking oversight dataset on stock transactions of officers of publicly traded companies.
  • Signatories of parliamentary petitions, as released by the previous version of the Bundestag's petition system. A historic database for research is available upon request.
  • Administrative boundaries (Kreise), web-ready SVG vector maps based on the official BKG data.

United Kingdom

  • Central Government Monthly Expenditures (in cooperation with data.gov.uk), aggregating detailed transactional expenditure data from all central government authorities into a single dataset.
  • Whole of Government Accounts (WGA), attempts to decipher the audited accounts section of the UK's COINS database, which involves a particularly nasty Excel sheet.
